联系我们

发表于 2016-08-19 分类于 Magento 2 用户使用手册

原文地址

店铺页面底部的联系我们(Contact Us)链接是客户和您保持联系的简易方式。客户可以通过填写表单来给您发送消息。

表单提交后，客户会看到一个感谢信息。该表单包含在contact-us-info block 中，您很容易就可以修改他。

配置 联系我们

  1. 后台侧边栏，点击店铺(Stores) > 设置(Settings) > 配置(Configuration)

  2. 在左侧面板选择常规(General)> 联系人(Contacts)

  3. 展开 联系我们(Contact Us)，有必要的话，请设置启用联系我们(Enable Contact Us)为“是”

  4. 展开邮箱选项(Email Options)做如下操作：

    a. 在发送电子邮件至(Send Emails to)字段中，输入联系我们表单将发送邮件的电子邮件地址
    b. 设置电子邮件发件人(Email Sender)。例如：自定义电子邮件
    c. 设置联系我们表单发送邮件用到的电子邮件模板(Email Template)

  5. 完成后，点击保存设置

定制联系我的表单

  1. 后台侧边栏，点击内容(Content) > 元素(Elements) > 页块(Blocks)
  2. 找到 Contact Us Info block ，打开编辑
  3. 下拉到联系部分，进行修改
    • 使用编辑工具格式化文字，加入图片和链接
    • 点击Show / Hide Editor 直接操作HTML
  4. 完成后，点击保存(Save Block)

店铺邮箱

发表于 2016-08-19 分类于 Magento 2 用户使用手册

原文地址

每个店铺(Store)或店铺视图(View)最多可以有5个不同的邮箱来表示不同的功能或部门。除了下面三个预定义的邮箱标识，剩下两个您可以自己定义。

  • 一般联系人(General Contact)
  • 销售代表(Sales Representative)
  • 客户支持(Customer Support)

每个标识和他对应得邮箱可以和特定的自动邮件消息联系起来，并显示为从您的店铺发送的电子邮件的发送者。

第一步：设置您域名的邮箱地址

在配置您的邮箱地址之前，每个邮箱地址都必须是您域名的有效地址。请您按照服务器管理员或电子邮件托管服务提供商提供的说明，创建需要的电子邮箱地址。

第二步：为店铺配置邮箱

  1. 后台侧边栏，点击店铺(Stores) > 设置(Settings) > 配置(Configuration)

  2. 在左侧面板选择常规(General)> 店铺电子邮箱地址(Store Email Addresses)

  3. 展开 一般联系人(General Contact) 做如下操作:

    a. 在发件人姓名(Sender Name)字段填入发件人的姓名，它将作为一般联系人的姓名显示在发送的邮件中。
    b. 发件邮箱(Sender Email)字段填入相关邮箱地址

  4. 重复上述过程给每个店铺设置好邮箱地址

  5. 完成后，点击保存设置。

第三步：更新销售电子邮箱的设置

如果您使用自定义的电子邮件地址，请务必更新相关的邮件配置，以正确显示发件人。

  1. 在左侧面板销售(Sales)下选择销售电子邮件(Sales Emails)页面有以下几个单独部分：
    订单和订单评论 (Order and Order Comments)
    发票和发票评论 (Invoice and Invoice Comments)
    装运和装运评论 (Shipment and Shipment Comments)
    贷项通知单和贷项通知单评论 (Credit Memo and Credit Memo Comments)

  2. 从 Order 开始，依次展开，选择合适的邮件发送者

  3. 完成后，点击 保存设置

货币

发表于 2016-08-19 分类于 Magento 2 用户使用手册

原文地址

货币设置


设置基础货币和接受的其他外币。建立用于自动更新货币汇率的导入连接和时间表。

货币符号


定义出现在商品价格和销售单据，如订单和发票里的货币符号。Magento支持来自世界各地超过200个国家的货币。

更新货币汇率


汇率可以手动进行更新或根据需要，或根据预定的时间表导入到店铺中。

货币选择器


如果有多种货币可供选择，店头会出现货币选择器。

商户位置设置用来配置付款方式(Payment methods)，如果没有设置任何值的话，会使用默认的国家(Default Country)

国家选项

发表于 2016-08-18 分类于 Magento 2 用户使用手册

原文地址

国家选项确定您开展业务所在的国家及您接受来自哪些国家的付款。

设置店铺的国家选项

  1. 后台侧边栏，点击店铺(Stores) > 设置(Settings) > 配置(Configuration)
  2. 在左侧面板选择常规(General)> 常规(General)
  3. 展开 国家选项 做如下操作
    a. 选择您业务所在地的默认国家(Default Country)
    b. 在接单国家(Allow Countries)列表中选择您接受订单的国家。默认情况下，所有国家都是选中的。（多选用Ctrl(PC)或者Command(Mac)）
    c. 在邮编可选(Zip/Postal Code is Optional for)列表中选择地址不需要邮编的国家
    d. 在最热门的目的地(Top Destinations)列表中，选择您销售的主要目标国家。
  4. 完成后点击保存设置

行政区域选项

发表于 2016-08-18 分类于 Magento 2 用户使用手册

原文地址

在很多国家，省、市或者区、州是邮寄地址的必要组成部分。该信息用来计算运输、开票方面的税率等。在没有这样要求的国家，行政区域字段可以被省略或者作为可选项。

由于各国对地址的格式要求大不相同，您还可以编辑用于发票，装箱单和运输标签上的格式化地址模板。

设置行政区域选项

  1. 后台侧边栏，点击店铺(Stores)> 设置(Settings)> 配置(Configuration)
  2. 左侧面板 常规(General)> 常规(General)
  3. 展开行政区域设置(State Options)，做以下操作：
    a. 在哪些国家需要行政区域设置(State is required for)列表中选择需要的国家
    b. 设置不强制要求的国家是否需要该字段(Allow to Choose State if It is Optional for Country)
    Yes 表示不强制要求必须填行政区域的，将该字段作为可选字段
    No 表示不强制要求必须填行政区域的国家，直接省略该字段
  4. 完成后，点击保存设置

区域设置选项

发表于 2016-08-18 分类于 Magento 2 用户使用手册

原文地址

区域决定了您整个店铺中使用的语言，国家，税率以及其他设置。区域设置选项决定了每个店铺的时区、语言，确定每周的营业时间。

设置区域选项

  1. 后台侧边栏，点击店铺(Stores)> 设置(Settings)>配置(Configuration)
  2. 左侧面板，选择常规(General) > 常规(General)
  3. 展开区域选项(Local Options)
  4. 从列表中选择时区(Timezone)，然后做以下操作
    a. 设置地区(Local)他决定了店铺前台的语言
    b. 设置重量单位，他是发货时用到的一般计量单位
    c. 设置每周第一天(First Day of the Week)
    d. 在休息日(Weekend Days)列表中进行选择（多选请使用Ctrl(PC)或者Command(Mac)）
  5. 完成后，点击保存设置

店铺信息

发表于 2016-08-18 分类于 Magento 2 用户使用手册

原文地址

店铺信息部分提供了用在销售文件和其他沟通中的基本信息

输入店铺信息

  1. 在后台侧标蓝选择店铺(Stores) > 设置(Settings) > 配置（Configuration）
  2. 左侧面板选择 常规(General) > 常规(General)
  3. 展开店铺信息(Store Information)部分，做下列操作
    a. 输入各种沟通途径中使用的店铺名称
    b. 输入店铺热线(Store Phone Number)，可以组织成您喜欢的格式
    c. 输入营业时间(Store Hours of Operation)。例如：周一至周五，9-5
    d. 选择您开展业务的国家(Country)
    e. 选择地区 ( Region/State)
    f. 输入店铺地址，如果地址很长，填入第二个地址栏(Store Address Line 2)
    g. 如果适用，输入VAT Number。要验证号码，点击Validate VAT Number按钮。
  4. 完成后，点击保存设置

管理后台

发表于 2016-08-18 分类于 Magento 2 用户使用手册

原文网址

您的店铺 Admin 是受密码保护的后台。在后台您可以设置产品，进行促销，管理订单以及执行其他管理任务。所有的基本设置事项和店铺管理操作都是由管理员完成的。

最初的登录凭证是在 Magento 安装过程中设置的，如果您忘记了密码，会发送临时密码到与管理员账户相关的邮箱中。为了增强安全性，您可以设置店铺登录的用户名和密码区分大小写。为了增强安全性，您可以设置管理员登录需要输入验证码。要了解更多，请参阅设置管理员账户安全

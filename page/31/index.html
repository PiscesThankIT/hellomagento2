操作控件

发表于 2016-08-18 分类于 Magento 2 用户使用手册

原文网址

与表格中的记录打交道，您可以通过操作控件对一条或多条记录进行操作。操作控件中列出了可对特定数据类型进行的操作。例如，对于商品记录，您可以用操作控件修改选中商品的属性，将其状态由”disabled”不可用的为”enabled”可用的，或者从数据库中删掉这条记录。

您可以做出尽量多的必要修改然后一次提交。这样比对每个商品分别修改要更快。

不同的列表有不同的操作控件列表，而且根据你所选择的操作命令，可能会出现更多选项。例如，修改一组记录的状态，操作控件边上会出现一个可选的状态选项。（此处2.1版本状态选项一直都在）

第一步：选择记录

第一列的复选框用来标记记录是否选中。过滤控件可以用来缩小范围。

  1. 勾选复选框来选中记录，或者使用批量工具(Mass Actions)来选择一组数据
    • 全选/全不选
  2. 如果需要，给每列设置筛选条件仅显示您要包含的记录

第二步：操作选中的记录

  1. 选择您要进行的操作
  2. 点击 提交(Submit)

例子：修改属性

  1. 列表中勾选上要修改的记录
  2. 点击 Actions 中的 修改属性(Update Attributes)然后点提交。修改属性页面列出了可以修改的属性，并且分组显示，左侧是他的分组。
  3. 选中修改(Change)复选框，然后进行修改。
  4. 保存
    .

表格布局

发表于 2016-08-18 分类于 Magento 2 用户使用手册

原文网址

您可以根据需要修改表格中展示的列，哪些展示哪些不展示，改变他们的顺序并保存为视图(Views)。默认情况下，显示20个中的9个（实际上是19个中的9个显示）

更改是否显示列

  1. 右上角，点击列(Columns)控件，然后可以做以下操作：
    • 要在表格中展示的打勾
    • 不要再表格中显示的去掉勾
  2. 记得下拉浏览所有可用的列

移动列

  1. 点击列的标题，按住不放
  2. 拖动到新的位置，然后松开

保存表格的视图

  1. 点击视图(View)点击保存当前视图(Save Current View)
  2. 输入视图的名称，然后点击箭头保存。当前视图名就变成您刚刚建立的那个视图了。

修改表格视图

点击视图(View)，然后做下列操作：

  • 点击视图名称切换另一个视图
  • 修改视图的名字，点击编辑(Edit)图标，进行修改。

后台页面通过表格控件展示数据集。每列顶部有控件可以用来给数据排序。
列标题的升序降序箭头表示当前的排序状态。您可以指定哪些列展示在表格中，还可以拖拽改变他们的位置。您还可以保存您对列的安排为视图留作以后用。操作列表列出了您可以对每条记录进行的操作。当前视图下的数据可以被导出为CSV 或者 XML 文件

  1. 点击过滤(Filters)
  2. 输入过滤条件
  3. 点击应用过滤（Apply Filters）

后台全局搜索

发表于 2016-08-18 分类于 Magento 2 用户使用手册

原文地址

头部的放大镜搜索框可以用来查找数据库中的任何记录，包括客户、商品、订单、相关属性。例如，你输入一个客户的名字，结果可能包含客户记录，可能也包含和这个名字相关的订单。

搜索

  1. 点击头部的放大镜图标打开搜索框，然后做以下操作：

    • 找相近结果，输入你要找的东西的开头几个字或字母
    • 找完全匹配的结果，输入你要找的东西的全称，可以是多个字词

  2. 在返回的搜索结果中，点击就可以查看记录了。

后台消息收件箱

发表于 2016-08-18 分类于 Magento 2 用户使用手册

原文地址

您的店铺会定期接收 Magento 的消息。消息内容可能是系统更新、补丁、发布了新版本、预定的维护、即将发生的事，而且这些消息被划分成不同的重要性级别。头部的响铃图标表示收件箱中未读邮件的数量。

至关重要的消息则会出现在弹出窗口中，您一登陆就会看到它。除非您将这类消息标记为已读或是删除掉，否则每次登陆都会提醒您。

通知列表中的消息按照严重程度、时间先后排序，越严重的越靠前，越新的消息越靠前。您可以通过操作命令标记一条消息为已读，查看详情或者从收件箱中移除他。

配置决定了收件箱更新的频次以及消息如何传输。如果您有一个安全的URL ，消息必须通过HTTPS进行传输。

查看收到的消息

  1. 点击头部的消息通知(Notification)图标，阅读摘要。您可以执行以下操作
    • 必要的话，点击消息，阅读全文
    • 点击消息右侧的删除图标，删除消息
    • 点击查看全部(See All)，进入通知列表
  2. 对于至关重要的消息，您可以执行以下操作
    • 点击阅读详情(Read Details)
    • 要关闭弹出窗口但保留消息，点击关闭(Close)

查看所有通知

  1. 可以做一下操作：
    • 点击头部的消息通知(Notification)，在消息盒子的底部点击查看全部(See ALl)
    • 侧边栏点击系统(System) > 其他设置(Other Settings) > 消息通知(Notifications)
  2. 在操作(Action)列，可以做以下操作：
    • 阅读详情请点击阅读详情(Read Details)，会在新窗口打开超链接。
    • 让消息留在收件箱，可以点击标记为已读(Mark As Read)
    • 要删除消息，点击删除(Remove)
  3. 作用于多条消息，您可以这样做：
    • 在第一列的多选框中打勾选中要操作的消息
    • 需要的话，可以用表头第一列的批量操作(Mass Actions)（多选框上方不起眼的图标）
  4. 选择以下操作中的一个
    • 标记为已读(Mark as Read)
    • 移除(Remove)
  5. 点击提交(Submit)完成操作

配置通知

  1. 在侧边栏，点击店铺(Stores)> 设置 (Settings) > 设置(Configuration)
  2. 下拉找到高级(Advanced) > 系统(System)
  3. 展开消息通知(Notifications)部分，做以下操作：
    • 如果您的后台运行在 HTTPS 下，那么设置 使用HTTPS获取订阅(Use HTTPS to Get Feed) 为是(Yes)
    • 设置更新频率(Update Frequency)决定消息收件箱的更新频率。时间间隔可以是1到24小时
  4. 完成后点击保存设置(Save Config)

管理后台菜单

发表于 2016-08-17 分类于 Magento 2 用户使用手册

原文地址
左边的侧边栏就是管理后台的主菜单了，专为台式机和移动设备而设计。弹出菜单提供了所有日常的店铺管理所需的工具。

控制面板
控制面板提供您店铺中销售和客户活动信息的简要概述，并且通常是您登录管理后台见到的第一个页面。

销售
在销售菜单里您可以找到关于订单处理，发票，发货，贷项通知单(credit memo)和交易(transitions)相关的一切。

商品
商品(products)菜单控制与商品目录和库存相关的一切。

客户
在客户菜单中您可以管理客户的账户，看到哪些客户正在线。

营销
在营销菜单中，您可以设置目录和购物车价格规则以及优惠券。满足特定条件时价格规则就会触发一些操作。

内容
内容菜单是您管理店铺内容和设计的地方。您将学习如何创建页面(pages)，内容块(blocks),前端应用程序，以及如何如何展示您的店铺。

报告
报告菜单提供的报告范围广泛，包括销售，购物车，商品，客户，标签，评论，和搜索字词，让您深入了解您店铺的每一个方面。

店铺
店铺菜单包含了配置和维护店铺各个方面的工具。

系统
系统菜单包括安装插件(extensions)，和管理与其他应用集成所需要的Web Services工具。

寻找合作伙伴&插件
在这里，您可以为您的店铺找到官方市场上的 Magento 合作伙伴和解决方案。

管理员账户

发表于 2016-08-17 分类于 Magento 2 用户使用手册

原文地址
您的管理员账户在安装时已经初步建好了，可能包含占位符信息还不不完整。您可以随时修改您的账户名、密码，更改姓名和邮箱地址。
要了解更多关于管理员用户(Admin users)和角色的，请参阅权限。

编辑您的账户信息：

  1. 管理后台右上角，点击「账户」(Account)图标，选择「账户设置」(Account Setting)
  2. 您可以对账户信息进行必要的修改了，如果您修改了登录所需的信息，一定要写下来以免忘记哦。
  3. 完成后，点击「保存账户信息」(Save Account)

Admin 登录

发表于 2016-08-17 分类于 Magento 2 用户使用手册

原文地址

您将学习的第一件事情是如何登陆和退出管理后台(Admin)，如何重置您的密码。该指南其余部分的说明都是为具有完整管理权限的用户所写，而且基于您已经成功登陆管理后台的假设。
要了解更多关于管理员用户(Admin users)和角色的，请参阅权限。

登录管理后台

  1. 在浏览器地址栏中输入安装时填好的 URL ，后面接上管理后台的基础路径。默认的管理后台 URL 长这样：http://www.yourdomain.com/admin
    您可以保存他为书签或者快捷方式放到桌面上，方便访问。
  2. 输入管理员「账户名」(User Name)和「密码」(Password)
  3. 点击登陆(Sign In)

重置密码

  1. 如果您忘记了密码，请点击「忘记密码」(Forgot Your Password?)链接
  2. 输入管理员账户的邮箱地址(Email Address)
  3. 点击「找回密码」(Retrieve Password)
    管理员密码必须大于等于7位，包含字母和数字，了解更多请参阅配置后台安全

退出管理后台

右上角点击「账户」(Account)图标，在弹出菜单里选择「退出」(Sign Out)

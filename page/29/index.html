这部分介绍店铺的后台管理和基本设置。你将了解以下概念：店铺层级(store hierarchy)和配置作用域(configuration scope)，并建立符合行业标准和要求的最佳实践。

网站(Websites), 店铺(Stores) 和 视图(Views)

发表于 2016-08-20 分类于 Magento 2 用户使用手册

原文网址

Magento 可以包括归一个 Admin 管理的多个网站，多个店铺，以及不同语言的视图。

网站(Websites)

Magento 2 安装后初始只有一个网站叫“主站”(Main Website)，您可以设置多个网站，每个网站拥有自己的域名。

店铺(Stores)

一个网站可以有多个店铺，每个店铺分别有自己的主菜单(main menu)。每个店铺可以有不同的商品和店面外观，不过都由同一个管理员管理。

视图(Views)

通常视图用来让一个店铺在不同的语言下展示。客户可以通过前台头部的语言选择器来选择不同的视图

单店模式

发表于 2016-08-19 分类于 Magento 2 用户使用手册

原文地址

如果您的 Magento 仅需要一个店铺一个视图，您可以关闭多店铺选项和范围提示做出简化。这份手册的大部分截图都是禁用单店模式情况下的，所以每个设置项下面都有范围提示。如果您后来添加了视图，那么单店模式就会被忽略。

设置单店模式

  1. 后台侧边栏，点击店铺(Stores)>设置(Settings)>配置(Configuration)
  2. 在常规(General)下，下拉到页面底部，展开单店模式(Single-Store Mode)
  3. 设置启用单店模式(Enable Single-Store Mode)为是
  4. 点击保存设置
  5. 刷新缓存

配置范围

发表于 2016-08-19 分类于 Magento 2 用户使用手册

原文地址
Magento 的安装有能力维护多个独立的网站(websites)，店铺(store)和店铺视图(store views)。如果您安装了多层级的网站、店铺和店铺视图，您可以选择将设置应用到上述哪个网站、店铺或是视图上。对于有权限限制的管理员用户，他只可以看到授权内的列表。

管理页面左上角的店铺视图选择器(Store view chooser)决定了您当前设置的作用范围。默认情况下，作用范围是默认配置(Default Config)

除非您的店铺运行在单店模式，不然每个设置字段下面浅灰色的小字表示了该字段的作用范围。如果您有多个网站、店铺或店铺视图，您应该每次都指定您设置作用的店铺视图。

设置项是[Store View]范围的，那么每个店铺视图的该设置项可以被设置成不同值。
不过有些设置是全局的[global]，比如邮编，因为整个系统都使用了同一个设置值。
[website]范围应用于网站级别，包括所有的店铺视图。

除了配置设置有范围，商品，目录，属性等也可以设置作用范围。了解更多，请查阅：产品作用范围

设置配置范围

在让设置作用于特定的网站、店铺或店铺视图前，做如下操作：

  1. 后台侧边栏，做以下操作：
    • 对于大部分的设置，点击店铺(Stores)>设置(Settings)>配置(Configuration)
    • 关于设计部分的设置，点击内容(Content) > 设计(Design) > 配置 (Configuration)在表格中选择您要应用的店铺视图
  2. 去往要修改的设置，做如下操作：
    a. 在左上角，选择您设置要作用的视图(store view)，如果弹出要您确认切换作用域的弹出框，点击OK
    每个设置项后面会出现复选框，也可能出现更多的附加字段。
    b. 取消使用系统值(Use System value)复选框前的勾，然后修改作用于该视图的设置值
    c. 重复上述过程完成对其他字段设置值的修改
  3. 完成后，点击保存设置(Save Config)

店铺演示提示

发表于 2016-08-19 分类于 Magento 2 用户使用手册

原文地址

如果您的店铺已经上线但是还在建设中，您可以在店面顶端显示一个演示提示，好让别人知道您的店铺还没有正式开张呢。开张的时候，移除他就行了。这就像门上挂的牌子，从”Closed”翻成”Open”。店铺演示提示的具体格式取决于店铺的主题(theme)。

设置店铺演示提示

  1. 后台侧边栏，点击内容(Content) > 设计(Design) > 配置 (Configuration)

  2. 在表格中找到您要修改的 Store view ，在 Action 列，点击编辑(Edit)

  3. 在其他设置(Other Setttings)下展开头部(Header)部分

  4. 下拉到底，设置显示店铺演示提示(Display Demo Store Notice)

  5. 完成后，点击保存配置(Save Configuration)

版权声明

发表于 2016-08-19 分类于 Magento 2 用户使用手册

原文地址

您店铺的每一页页脚都有版权声明。作为最佳实践，版权声明应该包括本年度，并注明贵公司是该网站内容的合法拥有者。

这个字符&Copy;可以用来插入版权符号，看例子：
长格式示例
Copyright &Copy; 2016 Magento, Inc. All rights reserved.
短格式示例
&Copy; 2016 Magento, Inc. All rights reserved.

修改版权声明

  1. 后台侧边栏，点击内容(Content) > 设计(Design) > 配置 (Configuration)

  2. 在表格中找到您要修改的 Store view ，在 Action 列，点击编辑(Edit)，做如下操作：

    a. 在其他设置(Other Setttings)下展开页脚(Footer)部分

    b. 在版权(Copyright)字段输入版权信息，这个字符&Copy;可以用来插入版权符号。

  3. 完成后点击保存配置(Save Configuration)

欢迎信息

原文地址

头部的欢迎信息扩展到包含登录客户的名称。在店铺开张前，请务必修改每个店铺视图的默认欢迎信息。

修改欢迎信息

  1. 后台侧边栏，点击内容(Content) > 设计(Design) > 配置 (Configuration)

  2. 在表格中找到您要修改的 Store view ，在 Action 列，点击编辑(Edit)，做如下操作：

    b. 输入欢迎文本(Welcome Text)

  3. 完成后点击，保存设置(Save Configuration)

  4. 提示您刷新缓存时，您就点击消息中的 缓存管理(Cache Management )链接。根据提示刷新缓存。

上传 Favicon

发表于 2016-08-19 分类于 Magento 2 用户使用手册

原文网址

Favicon 是 favorite icon 的缩略写法，用来指每个浏览器标签页上的小图标。根据不同的浏览器上，该图标也出现在地址栏URL前面。

一般 Favicon 是16 x 16 像素或者 32 x 32 像素。尽管不是所有浏览器都接受ICO, PNG, JPG, and SVG 格式，不过 Magento 全都接受。受到最广泛支持的格式是 ICO 。其他格式也可以使用，只是有些浏览器可能不支持。网上有很多免费的工具可以生成 ICO 图像，或者将其他格式的图像转成 ICO 格式的。

第一步：创建 Favicon

  1. 选择您喜欢的图片编辑工具创建16 x 16 像素或者 32 x 32 像素的Logo
  2. （可选）用工具转成ICO格式保存下来

第二步：上传 Favicon 到店铺

  1. 后台侧边栏，点击内容(Content) > 设计 (Design) > 配置(Configuration)

  2. 在表格中找到您要修改的 Store view ，在 Action 列，点击编辑(Edit)

  3. 在其他设置下展开HTML 头部(HTML Head)，做如下操作

    a. 点击上传

    b. 如果您要删除当前 favicon ，点击图标左下角的删除按钮，然后重新上传一个。

  4. 完成后点击保存配置

    第三步：刷新缓存

  5. 提示您刷新缓存时，您就点击消息中的 缓存管理(Cache Management )

  6. 在列表中，选中页面缓存(Page Cache)前的复选框，这时候他已经被标记为过期无效了。

  7. 设置 Actions 为刷新(Refresh)点击提交(Submit)

  8. 看效果请回到前台，按F5刷新浏览器

上传您的 Logo

发表于 2016-08-19 分类于 Magento 2 用户使用手册

原文地址

店铺头部的 logo 大小和位置是由店铺的主题(theme)决定的。您的 Logo 可以是GIF, PNG, JPG, 或 SVG 格式，从后台上传。Magento 样例的 Logo 是 SVG 格式的。它是一种可缩放的矢量图形格式。

Logo 图像在服务器的下列位置，这个位置的同名图像会作为主题的 Logo
实际的路径： app/design/frontend/[vendor]/[theme]/web/images/logo.svg
相对路径： images/logo.svg

如果您不知道 Logo 的大小，或者其他图像的大小，您可以在浏览器中右击图像，审查该元素。

除了店铺头部的 Logo ，Logo 还会出现在电子邮件模板和 PDF 形式发票中，还有其他销售文件中。电子邮件模板和 PDF 形式发票对 Logo 的大小要求不同，需要分别上传。了解更多，请查阅：Communications

上传 Logo

  1. 后台控制面板，点击内容(Content) > 设计(Design) > 配置(Configuration)

  2. 找到您要设置的视图，在 Action 列，点击编辑(Edit)

  3. 展开头部(Header)，做如下操作：

    a. 上传新的Logo，点击上传(Upload)，然后从您的电脑商选文件
    b. 输入 Logo 图像的宽度和高度
    c. 给 Logo 图像的 Alt 字段，输入您希望鼠标悬停在图像上时出现的文本

  4. 完成后，点击保存配置(Save Configuration)

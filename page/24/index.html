页面工作区包含一些工具，帮助您快速找到需要的页面，完成单个或多个页面的日常维护工作。您还可以在表格中快速编辑页面的属性。

快速编辑页面的属性

  2. 修改如下属性：

默认页面 Default Pages

发表于 2016-10-21 分类于 Magento 2 用户使用手册

原文网址

默认页面的配置决定着和Base URL 相关的着陆页，和相应的主页。他还决定着页面找不到时的返回错误页面，和面包屑导航到最顶端时。

配置默认页面

  1. 后台侧边栏，点击店铺(Stores) > 设置(Settings) > 配置(Configuration)
  2. 在左侧面板选择常规(General)> Web
  3. 展开 默认页面(Default Pages) 做如下操作

    a. In the Default Web URL field, enter the relative path to the folder in the Magento installation that contains the landing page. The default value is “cms.”
    (译者注：此处指默认的首页对应的URL 路径，虽然我们在浏览器中看到的URL 和此处设置的不同，但是通过访问该路径可以访问到默认的首页。当您输入网站地址后，首页实际上转到了该页面，但是没有改变浏览器的地址栏。)
    b. 设置CMS Home Page 为您的首页
    c. In the Default No-route URL field, enter the relative path to the folder in the Magento installation where the page is redirected when a “404 Page Not Found” error occurs. The default value is “cms/index/noRoute.”
    (译者注：404页面的路径，同上面的首页路径)
    d. 设置CMS No Route Page为您404 Not Found 页面
    e. 设置CMS No Cookies Page为客户禁用Cookie时看到的页面。
    f. 如果想要每个CMS 页面均包含面包屑导航，设置Show Breadcrumbs for CMS Pages为 Yes
  4. 完成后，点击 Save Config

Shipping Label Workflow

发表于 2016-10-17 分类于 Magento 2 用户使用手册

原文地址

Shipping labels 可以在发货的时候或者发货之后进行创建。Shipping Labels 以pdf 的形式保存，可以下载到您的电脑。

  1. 管理员提交 Shipping Label 请求。店铺管理员填写必要信息后提交请求。
  2. 请求被发送给物流公司。Magento 联系物流公司，在物流公司的系统中创建一个订单。每个包裹都是一个单独的订单。
  3. 物流公司返回Label和物流单号
    • 一批货但多个包裹返回多个Shipping Labels
    • 如果您对同一个shipping label 多次生成，物流单号并不会变化。
    • RMA 号码的退回物品，会使用新的单号代替原先的。
  4. 管理员下载并打印。shipping label 生成后，magento 系统会创建新的shipment，并且可以打印 shipping label 了。如果shipping label 由于连接或其他原因创建失败了，shipment 不会被创建。
    根据您浏览器的设置，pdf 文件可以被浏览和打印。每个 Label 在 pdf 中单独的一页进行呈现。

    要生成 shipping labels，UPS Type 必须被设置为 United Parcel XML

  6. 如果是 UPS XML 账户，做如下操作：
    a. 设置 UPS Type 为 United Parcel Service XML
    b. 输入如下认证信息：

    c. 设置 Mode 为 Live，这样会以安全的连接连接到 UPS 系统。Development 模式不会通过安全的连接发送数据。
    d. 验证 Gateway XML URL
    e. 设置 Origin of the Shipment 为发货的起始位置。
    f. 如果您与UPS 达成了其他费用协议，请设置 Enable Negotiated Rates 为 Yes，然后在 Shipper Number 中输入UPS分配给你的六位数字码。

  7. 在 标题(Title)字段中，输入结算时显示的名称。默认情况下，标题是 United Parcel Service

第三步：完成集装箱描述

  1. 设置请求包装类型(Packages Request Type)

    • Use origin weight (few requests)
    • Divide to equal weight (one request)

  2. 在 Container 字段中，指定用于装运的典型包装类型

  3. 设置重量单位(Weight Unit)。UPS 支持的重量单位因不同国家而不同。如果你不是很清楚，请询问UPS。支持的重量单位有：

发货条款(Shipping Policy)

发表于 2016-10-15 分类于 Magento 2 用户使用手册

原文网址

要在结账时显示您的发货条款，请完成发货条款参数的配置。

配置您的发货条款

  1. 后台侧边栏，点击店铺(Stores) > 设置(Settings) > 配置(Configuration)

  4. 完成后点击 Save Config
